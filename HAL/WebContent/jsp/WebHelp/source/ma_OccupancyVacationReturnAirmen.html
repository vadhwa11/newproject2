Occupancy / Vacation Return Airmen

Screen Overview
This screen will capture for Occupancy / Vacation Return Airmen.



Vacation Details:

SMQ:
User can select the SMQ from the given list, this will come from SMQ screen, it will display only airmen SMQ’s and occupied. It can't remain blank. Based on selection of SMQ all occupied details will display it contains Allotment order file number, allotment order file date, SMQ number, Pool code, Service number, service person name and rank.
Vacation No:
It is system generation number, user can’t change number.
Vacation Date:
It is system generated date, user can’t change date and it is present date.
Vacation:
User can check vacation selection button.


Allotment Details:

Allotment order file No:
It is system generation number, user can’t change number.
Allotment order file Date:
It is system generated date, user can’t change date and it is present date.
Service No:
User can enter service number, It can't remain blank.
Service Person Name:
User can enter service person details, It can't remain blank.
Poll Code:
User can select the Pool code from the given list, this will come from Pool screen, It can't remain blank.
Allotment Date:
It is system generated date, user can’t change date and it is present date.
HO/TO Date:
It is system generated date, user can’t change date and it is present date.
Save (Button):
By clicking on save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Order (Button):
Clicking on order button, user can print Allotment/Vacation order.
