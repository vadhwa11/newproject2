CRV (Certificate Receipt Voucher)

Screen Overview
This screen captures the details of CRV (Certificate receipt voucher) entry.



CRV Details

Source of Supply :
User can select source of supply from list. Source of supply can be Local purchase, CRV without LPO, PVMS by DGRC, AFMSD, AFMSD without indent & other units. This is mandatory.
CRV No. :
This is yearly auto generated no by the system. Exp- 01/09-10,02/09-10 etc. This is mandatory.
CRV Date :
Current date will be displayed by the system. User can change this date. This is mandatory.
Unit / Vendor Name:
Based on source of supply selection, Unit / vendor name will be filled. User can select vendor name from approved vendor name list. This is mandatory.
Indt No./ SO No.:
Indent no./ SO no. will be filled based on unit/ vendor name selection. User can select indent/ SO no. from list. If SO no. is selected , Get SO item button is displayed. By clicking on Get SO item button user can select list of items from supply order.
AT/SO No.:
User can enter AT/SO no.
AT/SO date :
User can select AT/SO date.
R/R No.:
User can enter R/R (Railway receipt) no.
Mode of conv. :
User can select mode of conveyance from list.
Unpac- Chk By :
User can select unpacked checked by from list.
How received:
User can enter how received details.
Remarks :
User can enter remarks.
Invoice No. :
User can enter invoice no. This is mandatory.
Invoice date :
User can select invoice date. This is mandatory.
Invoice Amount :
User can enter invoice amount. This is mandatory.
Freight duty :
User can enter freight duty.
Excise duty :
User can enter excise duty.
Octroi :
User can enter octroi.
Insurance charge :
User can enter insurance charge.
Other Charges :
User can enter other charges.
Custom duty :
User can enter custom duty.

CRV Details:

SR No. :
Auto sequence no. will be generated by the system.
PVMS No. :
User can enter PVMS No. and press tab. Details of item will be displayed.
Nomenclature :
It is description of PVMS No. If PVMS / NIV no. is not entered then type nomenclature initial 2-3 characters, list will be displayed. After nomenclature selection A/U, brand list and other details will be displayed.
Brand Name :
Select brand name.
Manuf. Name :
Select manufacturer name
A/U :
Accounting unit of PVMS No. Based on PVMS No. nomenclature , A/U will be displayed.
Batch No. :
Enter batch no. This is mandatory.
Quantity Received :
Enter actual received quantity. This is mandatory.
Free Qty :
User can enter free quantity .
Dispense type :
User can select dispense type from list. This is mandatory.
MDQ :
User can enter MDQ.( Minimum dispense quantity). This is mandatory.
Rate per MDQ :
User can enter rate per MDQ. This is mandatory.
ED for QR :
Data will be displayed .
Converted Stock :
Based on received qty., converted stock will be displayed.
Discount(%) :
User can enter discount in percentage or data will be displayed if it is mentioned in supply order.
TaxAmt /MDQ:
User can enter Taxamt per MDQ.
Amt. Value :
Amount will be displayed based on qty, rate, discount & tax.
Free Item :
User can select free item to Yes if current item is free with other item.
Manuf. Date :
Select items date of maufacurer.
Expiry date :
Select item expiry date.
Total VAT amount :
Enter VAT amount.
Total Discount :
Enter discount amount.
CRV Value :
CRV value will be displayed.
Round off Value :
Round off value will be displayed.
VAT Added :
Select VAT added if vat is already included in item wise.
Submit (Button) :
By clicking on submit button data will be saved.
PVMS/NIV (Button) :
By clicking on PVMS/NIV button PVMS/NIV master screen will be displayed.
Vendor (Button) :
By clicking on Vendor button Vendor master screen will be displayed.
