ME Scale

Screen Overview
This screen captures the details of ME scale.



ME scale number:
Select ME scale no. from the list. This is mandatory.
ME scale description:
ME scale description will be displayed.
Add (Button):
By clicking on add button new screen will be displayed for item addition.
Sr. No.:
By default auto serial no. will be displayed.
PVMS/NIV:
Write PVMS/NIV no. or based on nomenclature selection PVMS no. will be displayed.
Nomenclature:
Write nomenclature name.
A/U:
A/U will be displayed by the system.
Section:
Section will be displayed by the system.
Quantity:
Write ME scale quantity.
Submit (Button):
By clicking on submit button data will be saved.
