ED Return

Screen Overview
This screen captures the Excuse Duty information for the out patient. This screen will be used for maintaining information for both Officers /Airmen. This screen will contain information of days rest (ED) and Disposal.



From Date:
User can select from date, default it will show present date.
To Date:
User can select to date, default it will show present date.
Category:
User can select the category from the given list, this will come from category screen, it can remain blank.
Show (Button):
Based on from date, to date and category user can click show button to get the result in grid.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no., the he can simply enter the page no. in the given text box and click on Go Button.
Start Date:
User can select start date, it should be greater that visit date or same date.
No of days:
User can write no of days required.
Disposal:
User can select the disposal details from the given list, this will come from pre defined values, it can remain blank. If user select ED 2 days maximum allowed.
Diagnosis:
User can write present patient diagnosis.
Save (Button):
By clicking on save button user can save the data.
