IPD Ward Consumption

Screen Overview
This screen captures the ward drug consumption details. Use can enter drug usage for the particular period.



Details:
On clicking of ward consumption button user can access the ward consumption screen.
From Date:
User can select drug usage start date, it can’t remain blank.
To Date:
User can select drug usage end date, it can’t remain blank.
Department:
It is user login ward, it will show based on user login ward. User can’t change department.
Current Date:
It is system generated date and present date and user can’t change date, it can’t remain blank.
Time:
It is system generated time and present time and user can’t change time, it can’t remain blank.
View Quantity issued:
User can select previous issue number and user can view previous drug issue history, it can remain blank.
IP Issue No:
It is auto generation number, User can’t modify this number. And it can’t remain blank.
Page No:
Depends upon user entry page number will increase.
PVMS No:
It is ready only, user can’t select any PVMS no, it can’t remain blank.
Nomenclature:
User can select drug name from the given list based on user selection list will display. Based on user drug name PVMS no will come automatically, User can’t modify then PVMS number. Based on drug name another popup will display user can enter consumption quantity.
A/U:
Based on user drug name accounting unit (unit of measure) will come automatically, User can’t modify then accounting unit details..
Total Quantity issued:
Based on user entry quantity in popup screen total quantity will display here.
Next (Button):
By clicking on next button user can enter more drug names in the list. User can move to another page.
Back (Button):
Clicking on Back button, will move user to the previous screen.
Delete (Button):
By clicking on delete button user can delete the existing record.ata.
