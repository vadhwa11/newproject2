Work Not Completed with in 3 Days

Screen Overview
This screen captures the Work not completed with in 3 days user can enter PDC Date, PDC Time and PDC Remarks Details.





Complaint Code or Description:
On the basis of the search requirement, user can either choose complaint code or complaint description.
Textbox and Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
Department:
User can select the department from the given list, this will come from department screen, and it can't remain blank.
Complaint type:
User can select the complaint type from the given list, this will come from complaint type screen, and it can't remain blank.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Go Button.
Complaint Register:
When user clicking on grid the Complaint Number, Complaint Date, Complaint Time, Complaint type, Complaint Criteria, Service No, Service Person name, Department, Building no, Complaint Details will display. All are read only user canâ€™t change the existing value.
PDC Date:
User can write PDC Date details, it should be greater than Complaint Date, it can't remain blank.
PDC Time:
User can write PDC Time details, it can't remain blank.
C ADM O Remarks:
User can write Observations and remarks against the work. it can't remain blank.
Commandant Remarks:
Commandant can write Observations and remarks against the work.
Save (Button):
By clicking on save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Back (Button):
Clicking on Back button, user will move to the previous screen.
