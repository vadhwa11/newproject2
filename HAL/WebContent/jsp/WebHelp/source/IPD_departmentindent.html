IPD Department Indent

Screen Overview
This screen captures the details of inpatient department indent details.



Search:
User can select demand no from the give list based on demand number all drug issues list will display. User can modify the existing medicines list.
Demand No:
It is auto generation number, User can’t modify this number. And it can’t remain blank.
Demand Date:
If user waIt is system generated date and present date and user can’t change date, it can’t remain blank.
From Ward:
It is user login ward, it will show based on user login ward. User can’t change department.
To Ward:
User can select the toward from the given list, this will come from Department screen, it can't remain blank.
Approved By:
User can select the approved by from the given list, this will come from employee screen, it can't remain blank.
Requested By:
User can select the requested by from the given list, this will come from employee screen, it can't remain blank.
Import last indent (Button):
By clicking on import last indent button user can get previous medicine indent details will come automatically then user can update the existing records.
ADD (Button):
By clicking on add button user will get all nomenclatures user can add the items, it will shows stock in hand, loan received. User can write requested quantity, it can’t remain blank.
Submit (Button):
By clicking on Submit button user can save the selected data and it will back to main screen.
Delete (Button):
By clicking on Delete button user can delete the selected item.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Print (Button):
Clicking on print button, user can take department indent print out.
