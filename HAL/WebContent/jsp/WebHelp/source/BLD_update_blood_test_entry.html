Update Blood Test Entry

Screen Overview
By using this screen user can update blood test entry.





Search (Button):
User can search the records based on the Serial No.,Service No.,Name . Just click on the particuler record to which you want to update.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on OK Button.
Test Date:
User can update test date into this text box by clicking on the calender given on the side of the text box.
Service No:
User can update service number details.
Name:
User can update name details, it can't remain blank.
Rank:
User can update rank from the given list.
Relation:
User can update relation from the given list.
Patient Type:
User can update Patient Type from the given list.
Unit:
User can update unit from the given list, it can't remain blank.
Sex:
User can update sex from the given list, it can't remain blank.
Age:
User can update patient age details, it can't remain blank.
Contact No:
User can update Contact No details, it can't remain blank.
Received By drop Down Box:
User can update Received By person name from the given list, it can't remain blank.
GRID:
User can update different test name and results into corresponding boxes into grid.
Submit (Button):
By clicking on Submit button user can Save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
