CRV

Screen Overview
This screen captures the details of CRV for non expendable items.



CRV Details
Source of supply:
Select source of supply from the list.
CRV No.:
This is yearly auto generated no by the system. Exp- 01/09-10,02/09-10 etc. This is mandatory.
CRV Date:
Current date will be displayed by the system. User can change this date. This is mandatory.
Unit / Vendor Name:
Based on source of supply selection, Unit / vendor name will be filled. User can select vendor name from approved vendor name list. This is mandatory.
Indt No./ SO No.:
Indent no./ SO no. will be filled based on unit/ vendor name selection. User can select indent/ SO no. from list.
AT/SO date:
User can select AT/SO date.
Date rec/ surplus:
User can select date received surplus date.
R/R No.:
User can write R/R (Railway receipt) no.
Mode of conv.:
User can select mode of conveyance from list.
Unpac- chk by :
User can select unpacked checked by from list.
AT/SO No.:
User can write AT/SO no.
Technical Specification:
User can write item technical specification details.
AMC contract:
User can write AMC contract details.
How received:
User can how received details.
ME scale:
User can select ME scale.
Remarks :
User can write remarks.
Invoice No. :
User can write invoice no.
Invoice date:
User can select invoice date.
Invoice Amount:
User can write invoice amount.
Freight duty:
User can write freight duty.
Excise duty:
User can write excise duty.
Octroi:
User can write octroi.
Insurance charge:
User can write insurance charge.
Other Charges:
User can write other charges.
Custom duty:
User can write custom duty.
CRV Details: SR No.:
Auto sequence no. will be generated by the system.
PVMS No.:
User can write PVMS No. and press tab. Details of item will be displayed.
Nomenclature:
It is description of PVMS No. If PVMS / NIV no. is not selected then type nomenclature initial 2-3 characters, list will be displayed. After nomenclature selection A/U will be displayed.
A/U:
Accounting unit of PVMS No will be displayed.
Serial No.:
User can write serial no.
Accepted model:
User can write accepted model.
Lot No.:
Write Lot no. This is mandatory.
Quantity Received:
Write actual received quantity. This is mandatory.
Unit Rate:
User can write unit rate. This is mandatory.
Discount (%):
User can write discount in percentage.
Tax (%):
User can write Tax.
Amt. Value:
Amount will be displayed based on qty, rate, discount & tax.
Free qty.:
User can write free qty.
Manufacturer name:
User can select manufacturer name from the list.
Free Item:
User can select yes if item is free.
Installation date:
Select item installation date.
AMC start date:
Select AMC start date.
AMC end date:
Select AMC end date.
Warranty end date:
Select warranty end date.
CRV Value:
Sum of all item amount will be displayed.
Submit (Button):
By clicking on submit button data will be saved.
Add PVMS/NIV (Button):
By clicking on PVMS/NIV button PVMS/NIV master screen will be displayed.
