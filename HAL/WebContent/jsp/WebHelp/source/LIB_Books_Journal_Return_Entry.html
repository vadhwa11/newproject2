Books/Journal Return Entry

Screen Overview
This screen captures the Library Books/Journal Return Entry details.



Return No.:
This Field is autogenerated.
Return Date:
User can enter Return Date in this by clicking on the calender given on the side of the box.
Issue No:
User can enter Issue Number details, it can't remain blank.
Issue Date:
User can enter Issue Date details.
Service Number:
User can enter Service Number details.
Name:
User can enter Name details.
Rank:
User can enter Rank details.
Received By:
User can select Received By from the given list, it can't remain blank.
Sl No.:
This will show the serial number details.
Acc. No.:
This will show the accession number details.
Name of the Book:
This will show the Name details.
Quantity:
This will show the quantity details.
Return Check Box:
Select the records by clicking on these check boxes.
Submit (Button):
By clicking on Submit button user can Save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
