Defective Drugs Entry

Screen Overview
This screen captures the details of defective drugs entry. This entry is required when expiry drugs details is sent to DGAFMS.



Defect No.:
This is yearly auto generated no by the system. Exp- 01/09-10,02/09-10 etc. This is mandatory.
Entry Date:
Current date will be displayed by the system. This is mandatory.
From store:
By default login department name will be displayed.
To Place:
By default HQTCIAF(DPMO), JC Nagar Post, Bangalore – 560006 will be displayed.

Defective Drugs Detail:
SR No.:
Auto sequence no. will be generated by the system.
PVMS No.:
User can enter PVMS No. and press tab. Details of item will be displayed.
Nomenclature:
It is description of PVMS No. If PVMS / NIV no. is not entered then type nomenclature initial 2-3 characters, list will be displayed. After nomenclature selection A/U, brand list and other details will be displayed.
A/U:
Accounting unit of PVMS No will be displayed.
Batch no. :
All available Batch no. will be displayed in list. Select expiry batch from the list.
Brand Name:
Select brand name from the list.
Manufacturer:
Manufacturer name will be displayed automatically.
Expiry date :
Expiry date.will be displayed based on batch no.
Quantity :
Available quantity will be displayed.
Authy.
User can enter authy. Details.
Disposal:
User can enter disposal details.
Remarks:
User can enter remarks.
Desc1.:
User can enter description.
Desc2:
User can enter description.
Desc3:
User can enter description.
Copy To:
By default following value will be displayed… AFMSD, Kandivli East , Mumbai LAO(AF), Vimanpura, Bangalore Sr. Accts. Officer , CHAF
Submit (Button) :
By clicking on submit button data will be saved.
Adjust(Button) :
By clicking on adjust button data will be adjusted.
Print (Button) :
By clicking on Print button document will be printed.
