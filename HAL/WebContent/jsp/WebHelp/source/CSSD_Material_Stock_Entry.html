Material Stock Entry

Screen Overview
This screen have list of Material Stock Entry.



Entry No. Drop Down Box:
User can Select Entry Number from the given list, it can't remain blank.
Department:
Default login department will display and user can't modify the value.
Approved By:
User can select the approved by from the given list, this will come from employee screen, it can't remain blank.
Material Code:
It will dispaly automatically based on material name and it can't remain blank.
Material Name:
User can write the Material Name in the given box and it can't remain blank.
Quantity Text Box:
User can write Quantity of the material and it can't remain blank.
Remarks Text Box:
User can write Remarks details and it can remain blank.
Stock Entry Details (Grid):
This grid will show the list of all materials added.
Add (Button):
By clicking on Add button user can save the data.
Go Page Button and Text Box:
If user wants to switch directly to any particular page no., the he can simply enter the page no. in the given text box and click on Go Button.
