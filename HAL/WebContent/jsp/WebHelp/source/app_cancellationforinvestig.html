Cancellation For Investigation Appointments

Screen Overview
This screen captures the cancellation for Investigation Appointments for the patient. This screen will be used for maintaining Cancellation for Investigation appointments information for all clinical departments. This screen will contain information of Service no, HIN No, Department, Patient Name, Age, Appointment date, appointment No and equipment.



Patient Search:
User can enter Service no, HIN No, Department, Patient Name, Age, Appointment date, appointment No and equipment. , it can remain blank.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five record will be displayed. To navigate or look for the next record a pagination (the system of numbering pages ) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no., the he can simply enter the page no. in the given text box and click on Go Button.
Submit (Button):
By clicking on Submit button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Cancel Appointments (Button):
User can select any patient and user can cancel the appointments
