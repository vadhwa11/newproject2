OPD Urology Case Sheet

Screen Overview
This screen will capture Urology patient case sheet details.



Service Personnel Details:
Based on selection of patient all service personal details will come from MI Room.
Patient Details:
Based on selection of patient all patient details will come from MI Room.
Symptomme:
User can write patient symptommes and patient problems.
Haematuria:
User can select one or more values.It contains 4 types.
1) Total 2) Terminal 3) Initial 4) Clots.
Pain:
User can select one or more values.
It contains Left value and Right value. Left and right contain 4 values user can select one or more values.
1) Hypogastric pain, 2) Bone Pain 3) Dysuria 4) Perineal / Rectal
LUTS (Irritative):
User can select one or more values.It contains 4 values
1) Frequency, 2) Nocturia 3) Urgency 4) Urge incontinence
LUTS (Obstructive):
User can select one or more values.
It contains 4 values
1) Hesitanvy, 2) Straining 3) Sense of incomplete evaluation 4) Intermittency 5) Dribbling
Calcularia:
User can select calcularia and write observations.
Chyluria:
User can select Chyluria and write observations.
Erectile Dysfunction:
User can select erectile Dysfunction and write observations.
Ejaculatory Dysfunction:
User can select one or more values.
It contains 4 values
1) Haemospermia, 2) Premature Ejaculation 3) Retrograde Ejaculation 4) Anejaculation Ejaculation
Incontinence :
It contains 4 values
1) Stress Incontinence 2) Urge incontinence 3) Total incontinence 4) Overflow incontinence
Acute Urinary Retention:
User can select acute Urinary retention and write observations.
Submit (Button):
By clicking on submit button user can save the data.
View (Button):
By clicking on view button user can view the previous visit details.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
