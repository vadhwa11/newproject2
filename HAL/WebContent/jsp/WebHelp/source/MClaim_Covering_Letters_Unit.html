Covering Letters to Unit

Screen Overview
By using this screen user can submit Covering Letters to Unit.



Entry No.:
This will show the entry no.
Entry Date Text Box:
User can write Entry Date into this text box, this is mendatory field.
Unit Name Text Box:
User can write Unit Name into this Text Box, this is mendatory field.
Dispatch Date Text Box:
User can write Dispatch Date into this text box, this is mendatory field.
Unit Address Text Box:
Unit address text box will be filled according to the selection of the unit name.
Reimbursement of medical claim pending for dispatch Section:
This section will show the details of those patients for which we have submitted Contingent Bill for Reimbursement of Medical Bill Entry and where the amount is less then 10000/-.
Select check Box:
User can select the patient by checking on the check boxes for which he want to submit the covering letter.
Service No. Text Box:
This text box will show service number.
Rank Text Box:
This text box will show rank.
Name Text Box:
This text box will show name.
Amount Text Box:
This text box will show amount.
Submit (Button):
By clicking on Submit button user can Submit the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
