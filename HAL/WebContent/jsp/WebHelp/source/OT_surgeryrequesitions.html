Surgery Requisition for Patient

Screen Overview
This screen will capture patient surgery requisition for various departments.



Patient Search:
On the basis of the search requirement, user can choose service number, HIN number, Service Person first name, Service Person middle name, Service Person last name, patient first name, patient middle name, and patient last name.
Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five records will be displayed. To navigate or look for the next record a pagination (the system of numbering pages) will be there. User can select one patient from the grid.
OK Button and Text Box:
If user wants to switch directly to any particular page no., the he can simply enter the page no. in the given text box and click on Go Button
Service Personnel Details:
On the basis of user selection patient service personnel details will appear and user can’t modify data.
Patient Details:
On the basis of user selection patient details will appear and user can’t modify data.
Department:
User can select the department from the given list, this will come from department screen, it can't remain blank.
Surgeon Name:
User can select the doctors name from the given list, it can't remain blank.
Submit (Button):
By clicking on Submit button user can save the data.
