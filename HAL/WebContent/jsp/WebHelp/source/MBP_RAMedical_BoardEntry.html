Result of Appeal Medical Board Entry

Screen Overview
This screen will show the Result of Appeal Medical Board Entry.



Entry No.
This will show the entry no. this is autogenerated
Entry Date Text Box:
User can write Entry Date into this text box, this is mandatory field.
Batch No. Text Box:
User can write Batch No. into this text box, this is mandatory field.
Chest No. Text Box:
User can write Chest No. into this text box, this is mandatory field.
Name Text Box:
User can write Name into this text box, this is mandatory field.
Appeal Medical Board has Examined Text Box:
User can write Appeal Medical Board has Examined into this text box, this is mandatory field.
Add (Button):
By clicking on this button a new row will be inserted into this section for Unfit Explanation.
Remove (Button):
By clicking on this button a row will be deleted from this section.
Sr. No. Text Box:
This will show the SR. No. of the row and it is autogenerated field.
Unfit Explanation Text Box:
User can write Unfit Explanation into this text box
Submit (Button):
By clicking on Submit button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Search (Button):
By clicking on the search button user can again visit the search page.
