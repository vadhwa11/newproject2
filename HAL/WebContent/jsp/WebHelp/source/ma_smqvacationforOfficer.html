SMQ Vacation for officers

Screen Overview
This screen will capture for vacation of SMQ for officers.



Pool Code:
User can select the pool code from the given list, this will come from pool code screen, it will display only airmen pool wise .It can't remain blank. Based on selection of pool code all occupied SMQ’s list will display.
Vacation No:
It is system generation number, user can’t change number.
Vacation Date:
It is system generated date, user can’t change date and it is present date.
GRID (Display):
Based on search criteria data will display in the form of table format. All occupied SMQ’s details will display with allotment order file number, allotment order file date, SMQ number, pool code, Service No, Service person name, allotment date And HO/TO date.
Cancel:
User can select one or multiple SMQ numbers.
Save (Button):
By clicking on save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Vac.Order (Button):
Clicking on vacation order button, user can print vacation order details.
