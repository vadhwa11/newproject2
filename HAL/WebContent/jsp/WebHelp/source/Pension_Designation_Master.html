Designation Master

Screen Overview
By using this screen user can add or edit any Designation.



Search (Button):
User can search the records by writing Designation Code or Designation into the given text box just select the particuler option from the option buttons and click on the search button.

GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on OK Button.
Designation Code Text Box:
User can write Designation Code into this text box,this is mendatory field.
Designation Text Box:
User can write Designation in this text box, this is mendatory field.
Group Type Drop Down Box:
User can select Group Type from this drop down box, this is mendatory field.
Type Drop Down Box:
User can select the Type from this drop down box, this is mendatory field.
Add (Button):
By clicking on Add button user can save the data.
Reset (Button):
Clicking on reset button will clear the current or existing data.
