IPD Intake Output

Screen Overview
This screen captures the details of all patient IPD clinical chart entry details based on user login ward. User can enter only selected patient.



Patient Details:
On clicking of intake/output button user can access intake output screen for patient treatment details entry. Based on patient selection from main screen patient details will display user can’t modify patient details.
ADD ROW:
User wants to give multiple times user can create multiple rows.
DELETE ROW:
User wants to delete the present row user can delete the selected row.
Date:
It is system date, user can’t modify this date.
Time:
It is system time, user can modify this time.
Temperature:
User can select the temperature from the given list, this will come from free defined list and it can remain blank.
Pulse:
User can write present patient pulse details, it can remain blank.
Respiration:
User can write present patient respiration details, it can remain blank.

User can write present patient BP details, it can remain blank.

User can write present patient bowel details, it can remain blank.
Pain:
User can select the pain from the given list, this will come from free defined list and it can remain blank.
FHR:
User can write present patient FHR details, it can remain blank.
Remarks:
User can write present patient remarks or observations, it can remain blank.
