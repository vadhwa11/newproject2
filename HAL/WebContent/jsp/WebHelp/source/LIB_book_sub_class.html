Book Sub Class Master

Screen Overview
This screen have list of Books sub class a long with their sub class code, class name and status.



Sub Class Code or Sub Class Name:
On the basis of the search requirement, user can either choose Sub Class Code or Sub Class Name
Textbox and Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Go Button.
Sub Class Code:
User can write the Sub Class Code details, it can't remain blank.
Sub Class Name Text Box:
User can write the Sub Class Name details, it can't remain blank.
Class Name:
User can select the Class Name from the given list, this will come from Class Name screen, it can't remain blank.
Add (Button):
By clicking on Add button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Update (Button):
By Clicking on Update button, user can update the data.
Inactivate (Button):
By Clicking on Inactivate button, user can inactivate any machine item.
