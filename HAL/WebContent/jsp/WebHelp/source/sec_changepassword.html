Change Password

Screen Overview
This screen captures the user password change details.



Login Name:
User can write his login name based on login name employee name will display and it can't remain blank.
Old Password:
User can write his old password, it can't remain blank.
New Password:
User can write his new password, it can't remain blank.
Confirm Password:
User can write his confirm password, it can't remain blank. New password and confirm password should be same.
Submit (Button):
By clicking on submit Button user can submit the data.
Password Reset (Button):
Clicking on password reset button automatically old password will set as a current password.
