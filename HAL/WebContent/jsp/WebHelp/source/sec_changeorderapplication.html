Change Order for Application

Screen Overview
This screen captures the change the application display order.



Hospital Name:
User can select the hospital name from the given list, this will come from hospital screen, it can't remain blank.
Application Group:
User can select the application group from the given list, this will come from application group screen, it can't remain blank.Based on selection of application group all menu list will display.
Move Up (Button):
Based on selection of application id user can move to up or down. Clicking on move Up button, application will move to priority.
Submit (Button):
By clicking on submit Button user can submit the data.
Back (Button):
Clicking on Back button, will move user to the previous screen.
