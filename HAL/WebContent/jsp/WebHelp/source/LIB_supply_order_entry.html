Supply Order Entry

Screen Overview
This screen captures the Library supply order details.



Supply Order No:
It is auto generated number, it can't remain blank.
Date:
It is sytem generated date, it can't remain blank.
Vendor Name:
User can select the Vendor Name from the given list, this will come from vendor master screen, it can't remain blank.
Quotation No:
User can write Quotation No details, it can't remain blank.
Quotation Date:
User can write date in this text box by clicking on the calender given on the side of the box, it can't remain blank.
Remarks:
User can enter Remarks details
Add Row (Button):
User can Add a new row by clicking on this button to add the Books detail.
Delete Row (Button):
User can Delete the row by clicking on this button.
Book Name:
User can write Book Name in this box, it can't remain blank.
Author:
User can write Author in this text box, it can't remain blank.
Publisher:
Based on Book name Publisher details will come automatically.
Year Of Publication:
User can write Year Of Publication details.
Comparitive Cost:
User can write Comparitive Cost details, it can't remain blank.
Edition:
User can write book Edition details.
SAVE (Button):
By clicking on Save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
