Medical Library Module

Module Overview
Command Hospital Air Force Bangalore (CHAFB) needs to maintain the Medical Library, Books category, Books/journal supply order entry, CRV entry , Issue entry etc. efficiently at their hospital. At any given time the Medical Library Module should be able to get the Books pending report, Books purchase report, receipt report, Issue report & other reports.

In this Module has following Sub Modules
  1. Library Master
  2. Supply Order Entry
  3. Update Supply Order Entry
  4. CRV Entry
  5. Update CRV Entry
  6. Journal Receipt Register
  7. Update Journal Receipt Entry
  8. Book/Journal Issue Entry
  9. Update Book/Journal Issue Entry
  10. Book/Journal Return Entry
  11. Update Book/Journal Return Entry
  12. Book Stock Taking Entry
  13. Update Book Stock Taking Entry
