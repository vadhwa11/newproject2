Diet Indent Item Formula

Screen Overview
This screen maintains the details of item which is used in indent to Logistics department from Patient mess.



Item Name:
On the basis of the search requirement, user can write Item Name.
Textbox and Search (Button):
Based on search criteria data will display in the form of table. At a time only five records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no., the he can simply enter the page no. in the given text box and click on Go Button.
Diet Combination:
User can select the diet combination from the given list, this will come from Diet combination screen, it can't remain blank.
Item name:
User can select the item name from the given list, this will come from diet item name screen, it can't remain blank.
Qty:
User can write required quantity, it can't remain blank.
UOM:
User can select the Unit of measure from the given list, this will come from UOM screen and it can't remain blank.
Patient category:
User can select the patient category from the given list, it contains officers and general and it can't remain blank.
Validity Start Date:
User can write validate start date, it can't remain blank.
Save (Button):
By clicking on save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
