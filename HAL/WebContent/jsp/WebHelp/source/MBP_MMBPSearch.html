Medical Board Proceedings Search

Screen Overview
By using this screen user can search Medical Board Proceedings entry ,user can add or update any record for this just click on the particuler record from the GRID then the Medical Board Proceedings entry page will be visible from here user can do any Medical Board Proceedings entry.






Search (Button):
User can search the records based on the Service No.,Rank,Ad No.,First Name,Last Name,Just write your search criteria and click on the search button.

GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on OK Button.
Medical Board Proceeding Entry (AFMSF 15) Page:
this page will show the details of record selected by you from the GRID like - Entry No.,Entry Date,Hin No.,Ad No.,Name,Service No.,Rank,Unit/Ship,Service,Age,Years,Sex,Army/ Corps/ Branch/ Trade,Ceased Duty on etc.
