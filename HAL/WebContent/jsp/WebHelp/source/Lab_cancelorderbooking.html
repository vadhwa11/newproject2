Cancel Order Booking

Screen Overview
This screen captures the Cancel order booking for investigation of the patient. This screen will be used for maintaining Cancel order booking for investigations.



Search (Button):
User can search the records based on Order booking from date, Order book to date, Patient type, Service No, HIN No, A&D No, Service Person Name and Patient Name.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Go Button.
Test Details (Display):
All test details will display based on order number. User can select single test or multiple tests
Submit (Button):
By clicking on submit button user can save the data.sts.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Back (Button):
Clicking on Back button, user will move to the previous screen.
