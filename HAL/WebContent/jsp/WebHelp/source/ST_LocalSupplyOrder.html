Local Supply Order

Screen Overview
This screen captures the details of local supply order entry.



Budget status (Button):
By clicking on budget status button Current budget status is displayed with Total budgeted amount, Total SO committed amount and other details.
Approval Authority (Button):
By clicking on approval authority button It displays all approval authority name for selection. User can select one or more.
FileNo :
User can enter file no. for SO. This is mandatory.
S.O. Number :
This is yearly auto generated no by the system. Exp- 01/09-10,02/09-10 etc. This is mandatory.
S.O. Date :
Current date will be displayed by the system. User can change this date. This is mandatory.
Vendor Name:
User can select vendor name from approved vendor name list. This is mandatory.
Delivery date:
User can enter goods delivery date. This is mandatory.
Quotation No.:
User can enter quotation no. This is mandatory.
Quotation date :
User can select quotation date. This is mandatory.
Payment Terms :
User can select payment terms after clicking on select link or enter new payment terms .
Delivery terms :
User can select delivery terms after clicking on select link or enter new delivery terms .
Remarks :
User can enter remarks.
Signing authority :
User can enter signing authority.
Approval of CFA/IFA vide :
User can enter CFA/IFA vide.

Supply Order Details:

PVMS No. :
User can enter PVMS No. and press tab. Details of item will be displayed.
Nomenclature :
It is description of PVMS No. If PVMS / NIV no. is not entered then type nomenclature initial 2-3 characters, list will be displayed. After nomenclature selection A/U, brand list and other details will be displayed.
A/U :
Accounting unit of PVMS No. Based on PVMS No. / nomenclature , A/U will be displayed.
Actual quantity :
User can enter actual request quantity.
Free qty :
User can enter free quantity .
Dispense type :
User can select dispense type from list. This is mandatory.
MDQ :
User can enter MDQ.( Minimum dispense quantity). This is mandatory.
Ordered quantity :
User can enter actual SO order quantity. This is mandatory.
Rate per MDQ :
User can enter rate per MDQ. This is mandatory.
Total rate for OQ :
User can enter total rate for ordered quantity.
Excise duty for OQ :
User can enter excise duty for ordered quantity.
Discount(%) :
User can enter discount in percentage.
MRP :
User can enter MRP
Tax for OQ :
User can enter tax for ordered quantity.
Total Amount :
This is display field. It is calculated based on ordered qty, rate , discount & tax.
Free Item :
User can select free item to Yes if current item is free with other item.
Total amount :
Sum of all item amount will be displayed.
Delete Row (Button) :
By clicking on delete row , selected row from supply order details grid will be deleted.
Save & Next (Button) :
By clicking on save & next button current screen data will be saved and user can enter another 10 item for SO in same supply order no.
Submit (Button) :
By clicking on submit button data will be saved.
PVMS/NIV (Button) :
By clicking on PVMS/NIV button PVMS/NIV master screen will be displayed.
Search :
By clicking on search , user can select previous saved data.
