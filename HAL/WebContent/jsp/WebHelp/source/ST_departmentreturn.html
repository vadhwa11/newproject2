Department Return

Screen Overview
This screen captures the details of item returned to selected department.



Department Return

Return No.:
This is yearly auto generated no by the system. Exp- 01/09-10,02/09-10 etc. This is mandatory.
Return Date:
Current date will be displayed by the system. This is mandatory.
Reference No.:
Enter ref no. if available.
From department:
Login department name will be displayed.
To department:
Select department name from the list to whom you want to return. This is mandatory.
Return by:
Select return by from the list.
Received by:
Select received by from the list.
Reason:
Enter reason for return.
Remarks:
Enter remarks if any.

Item Details:

SR No. :
Auto sequence no. will be generated by the system.
PVMS No. :
User can enter PVMS No. and press tab. Details of item will be displayed.
Nomenclature :
It is description of PVMS No. If PVMS / NIV no. is not entered then type nomenclature initial 2-3 characters, list will be displayed. After nomenclature selection A/U, brand list and other details will be displayed.
Total qty returned :
Enter returned quantity.
Submit (Button) :
By clicking on submit button data will be saved.
