Donor Blood Pending For Sample Screening

Screen Overview
This screen captures the Donor Blood Pending For Sample Screening details of patients.






Search (Button):
User can search the records based on the From Date(This is mendatory field), To Date(This is mendatory field), Patient Name,Service No., HIN No., Doantion No.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on OK Button.
Service Personnel Details section:
This section show the following details - Service Type, Service No.,Service Status,Relation,Ser. Per. Name,Rank,Blood Bag No.,Donation No.,Sample Test No.,Sample Test Date,Sample Test By(Drop Down box.)
Sample Test By Dropn Down Box:
User can select the name of the sample test by person fron the dropn down box.
Test Name Text box:
User can write test name based on user entry all test list will display then user can select test name from the given list.
Result Drop Down Box:
User can select result positive/negative from the list.
Add (Button):
By clicking on Add button user can Save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
