Issue of Willingness Certificate

Screen Overview
This screen will capture issues of willingness certificate for officers.



Pool Code:
User can select the pool code from the given list, this will come from pool code screen, it will display only officer’s pools. It can't remain blank. Based on selection of pool code all vacant SMQs list and pending register officers list will display.
GRID (Display):
Based on search criteria data will display in the form of table. All pending registration details will display with Registration No, Registration Date, Service No, Service person name, And rank.
SMQ No:
User can select the SMQ No from the given list, this will come from SMQ screen, and it will display all vacated SMQ’s list. It can't remain blank.
Issue (Button):
By clicking on issue button user can print willingness certificate.
Accepted:
User can select from the list and issue those selected list only.
Save (Button):
By clicking on save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
