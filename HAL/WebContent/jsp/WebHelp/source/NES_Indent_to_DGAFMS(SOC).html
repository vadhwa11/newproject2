Indent to DGAFMS (SOC)

Screen Overview
This screen captures the details of indent to DGAFMS in the form of SOC. SOC is generated by the concerned department.



Indent No.:
Indent No. is generated automatically by the system.
Indent Date:
Current date is displayed as SOC date.
Command:
By default CHAF, Bangalore is displayed by the system.
Hosp/Unit Address:
Hosp/ Unit address will be displayed by the system.
Item req. for Dept./ward :

Nomenclature:
Type nomenclature initial 2-3 characters, list will be displayed. After nomenclature selection A/U, Stock in and other details will be displayed.
Brief mention of functions:
Write functions details.
IF PAC, General detail justification :
User can write general detail justification.
Dose (for drugs) Day/Course :
User can write dose for no. of days.
PVMS/NIV drugs substitute already prescribed :
User can write if drugs is already prescribed.
Justification of using NIV Item :
User can write justification if NIV item is prescribed.
Critical trial required :
User can select Yes/ NO if critical trial required.
Submit (Button) :
By clicking on submit button data will be saved.
Search :
By clicking on search , user can select previous saved data.
Print (Button) :
By clicking on print button report will be printed.
