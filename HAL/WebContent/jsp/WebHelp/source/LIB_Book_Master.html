Book Master

Screen Overview
This screen have list of Books.



Book acc No. or Book Name:
On the basis of the search requirement, user can either choose Book acc No or Book Name
Textbox and Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Go Button.
Book Acc No:
User can write the Book Acc Number detais, it can't remain blank.
Book Name:
User can write the Book Name details, it can't remain blank.
Book Category Name:
User can select the Book Category from the given list, this will come from Book Category screen, it can't remain blank.
Book Class Name:
User can select the Book Class from the given list, this will come from Book Class screen, it can't remain blank.
Book Sub Class Name:
User can select the Book Sub Class from the given list, this will come from Book Sub Class screen, it can't remain blank.
Author Name:
User can write the Book Author Name details, it can remain blank.
Year Of Publication:
User can write the Year Of Book Publication details, it can remain blank.
Date of Purchase:
User can write the Date of Purchase details, it can remain blank.
Edition:
User can write the Book Edition details, it can remain blank.
Location:
User can write the Location details, it can remain blank.
Type:
User can select the Book type from the given list, this will come from type screen, it can't remain blank.
Books For Department:
User can select the department from the given list, this will come from department screen, it can't remain blank.
Volume:
User can write the Volume details, it can remain blank.
Publisher Name:
User can select Publisher Name from the given list, it can remain blank.
Pages:
User can write the Pages in the book, it can remain blank.
Price:
User can write the Book cost Price, it can remain blank.
Add (Button):
By clicking on Add button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Update (Button):
By Clicking on Update button, user can update the data.
Inactivate (Button):
By Clicking on Inactivate button, user can inactivate any machine item.
