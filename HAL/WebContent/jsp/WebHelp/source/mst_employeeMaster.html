Employee Master

Screen Overview
This screen captures the employee details.



Service number, first name or last name:
On the basis of the search requirement, user can either choose service number, first name or last name.
Textbox and Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
Go Button and Text Box :
If user wants to switch directly to any particular page no., the he can simply enter the page no. in the given text box and click on Go Button.
