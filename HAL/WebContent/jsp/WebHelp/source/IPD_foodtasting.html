IPD Food Tasting

Screen Overview
This screen captures the details of ward food tasting details.



Tasting Observations:

Meals:
User can select meals type it contains Breakfast, Lunch and Dinner, it can remain blank.
Status:
User can select from the given list , it contains Good, Average and poor, it can remain blank.
Time:
User can write time of food tasting, it can remain blank.
Checked By:
User can select the food tasted from the given list, this will come from employee screen, it can remain blank.
Remarks:
User can write remarks of food tasting, it can remain blank.
Submit (Button):
By clicking on Submit button user can save the data.
Back (Button):
Clicking on Back button, will move user to the previous screen.
