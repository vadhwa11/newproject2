Leave Application Pending for Approval

Screen Overview
This screen captures the details of Leave Application Pending for Approval.



GRID:
This will show the details of the pending leaves applications.
Service no:
User can write service no. in this text box.
Name:
User can write name in this text box.
Rank:
User can write rank in this text box.
Leave From:
User can write leave from in this text box.
Leave To:
User can write leave to in this text box.
Total Leave Days:
User can write total leave days in this text box.
Approved:
User can select approved from the drop down box.
Approved BY:
User can select the approved by from the drop down box.
Approved Date:
User can write approved date in this text box.
Remarks:
User can write remarks in this text box.
Close:
By clicking on the close button user can close this form.
Submit:
By clicking on the submit button user can save the record.
