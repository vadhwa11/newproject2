Cancellation of Minor Work Proposal

Screen Overview
This screen will capture Cancel of Minor work Proposal details, if user want to cancel the existing proposals user can cancel.



Minor Work Proposal No search:
User can write existing proposal number and press tab all details will appear automatically.
Proposal Help (Button):
If user doesn’t the proposal number through help user can get information.
Minor Work Proposal No:
It will come from Minor work proposal screen, user can’t change number.
Minor Work Proposal Date :
It will come from Minor work proposal screen, user can’t change date.
Work Category:
It will come from Minor work proposal screen, user can’t change work category, and it can't remain blank.
Work type:
It will come from Minor work proposal screen, user can’t change work type, and it can't remain blank.
Type of Work:
User can modify existing details and write new details, it can't remain blank.
Department:
User can modify existing details and select new departments, it can't remain blank.
Justification:
User can modify existing details and write new details, it can't remain blank.
Save (Button):
By clicking on save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Back (Button):
Clicking on Back button, user will move to the previous screen.
