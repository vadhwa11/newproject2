Assign Button Rights to Employee Group

Screen Overview
This screen captures the assign form level button rights to employee group.



Hospital Name:
Default login hospital name will display. User can modify hospital name. Based on hospital selection all user groups list will display and user want to assign application rights user can select any group description.
Form Name:
User can select the form name from the given list, it can't remain blank.
Form Name:
User can select the form name from the given list, it can't remain blank.
Employee Group:
User can select the employee group from the given list it will come from user group screen, it can't remain blank. Based on user selection of hospital name, Form Name and Employee Group all button will display.
Button details grid:
Based on user selection of hospital name, Form Name and Employee Group all button will display automatically. User can select one button or multiple buttons.
Login User grid:
Based on user selection of hospital name and Employee Group all login user list will display automatically. User can select one or multiple user.
ADD Button Rights (Button):
Clicking on ADD Button Rights button, all button level rights will assign to selected users.
Back (Button):
Clicking on Back button, will move user to the previous screen
