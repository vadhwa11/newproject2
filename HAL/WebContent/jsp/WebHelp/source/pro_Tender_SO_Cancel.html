Tender Re SO For Cancelled

Screen Overview
This screen captures the details of tender re supply order for cancelled item.



Group:
Select group name from the list. Tender no. will be filled based on group selection.
Tender No.:
Select tender no.

Vendor Details:

SR No.:
Auto sequence no. will be generated by the system.
Vendor Name:
All cancelled item vendor name will be displayed by the system.
No. of Items:
Count of cancelled item for vendor will be displayed by the system.
Total Amount:
Sum of all L1 item amount will be displayed by the system.
Re Supply Order (Button):
By clicking on supply order button new screen will be displayed for supply order creation.
