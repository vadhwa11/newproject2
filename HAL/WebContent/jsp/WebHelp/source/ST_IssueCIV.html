Issue CIV IAFF(Q) 429

Screen Overview
This screen captures the details of CIV against indent/request.



Issue Details :

CIV No.:
This is yearly auto generated no by the system. Exp- 01/09-10,02/09-10 etc. This is mandatory.
Issue Date:
Current date will be displayed by the system. This is mandatory.
Issue To:
Select department/ward name from list. This is mandatory.
Demand date:
Demand date will be display based on demand no selection.
Request by:
Select request by from list.
Approved by:
Select approved by from list.
Issued by:
Select issued by from list. This is mandatory.
Demand No. :
User can select demand no. from list. Based on demand no. request item list will be displayed in grid.

Item details:

SR No. :
Auto sequence no. will be generated by the system.
PVMS No. :
PVMS No. will be displayed.
Nomenclature :
Nomenclature will be displayed.
A/U :
Accounting unit of PVMS No will be displayed.
Quantity Requested :
Enter actual received quantity. This is mandatory.
Available stock :
Current stock will be displayed for login department.
Loan out Qty :
Loan out qty. will be displayed if item is already issued on loan basis.
Qty. Issued(Button) :
On click of this button , issued screen will be displayed for issue qty. entry.
Submit (Button) :
By clicking on submit button data will be saved.
