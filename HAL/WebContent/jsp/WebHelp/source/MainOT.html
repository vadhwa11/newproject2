Operation Theater

Module Overview
This module deals with the full-fledged Operation Theater management system. The scope of the module covers OT Distribution entry, Surgery request by ward/OPD, , PAC (PRE-ANAESTHETIC ASSESSMENT) form entry, OT List preparation, Pre & Post Anaesthesia Procedure Notes entry, Anaesthesia register entry, Surgery procedure notes, Specimen dispatch entry, Human body parts disposal entry etc.

In this Module has following Sub Modules
  1. PAC Clearance List
  2. OT Booking List
  3. OT List Change
  4. Pre OP Surgery
  5. Post OP Notes (Anaesthesia)
  6. Post OP Notes (Surgery)
  7. Specimen dispatch Entry
  8. Human Body Parts Disposal
  9. Emergency OT Booking
  10. Specimen Dispatch Register
  11. View OT Pre-Anesthesia Details
  12. Reports
