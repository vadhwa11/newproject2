Technical Bid

Screen Overview
This screen captures the details of technical bid.



Group:
Select group name from the list. Tender no. will be filled based on group selection.
Tender No.:
Select tender no. This is mandatory.
Supplier/Vendor:
Select vendor name from the list. After vendor selection all item for which vendor has quoted will be displayed.
Manuf. Lab practices:
Select Yes if vendor has manufacturer lab practice certificate else no.
No conviction certificate:
Select Yes if vendor has no conviction certificate else no.
Mkt. standing certificate:
Select Yes if vendor has mkt. standing certificate else no.
Acceptance Lr Attached:
Select Yes if vendor has acceptance Lr attached else no.
PVMS/NIV:
If user wants to search for any PVMS or NIV item then write PVMS/NIV code and click on search button.

Item Details:

SR No.:
Auto sequence no. will be generated by the system.
PVMS No.:
PVMS No. will be displayed by the system.
Nomenclature:
Nomenclature will be displayed by the system.
Brand name:
Brand name will be displayed by the system.
Manufacturer name:
Manufacturer name will be displayed by the system.
A/U :
Accounting unit of PVMS No will be displayed.
Dispense Type:
Item Dispense type will be displayed.
MDQ:
MDQ will be displayed by the system
Rate per MDQ:
Rate per MDQ will be displayed by the system.
Items submitted:
Item submitted will be displayed by the system.
Manufacturing License:
Select Yes if vendor has manufacturing license for item.
Certificate to market:
Select yes if vendor has certificate to market for item.
Remarks:
User can write remarks.
Disqualify:
Select if vendor is not qualified for selected row item.
Update (Button):
By clicking on Update button changed data will be update.
Previous (Button):
By clicking on previous button 20 previous data will be displayed.
Next (Button):
By clicking on next button 20 next data will be displayed.
