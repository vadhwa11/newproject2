Human Body Parts Disposal Entry

Screen Overview
This screen will capture patient specimen dispatch entry details.



Entry No:
It is system generation number, User can’t change number and it can't remain blank.
Date:
It is system generation date, it is current date and it can't remain blank.
Service Personnel Details:
On the basis of user selection patient service personnel details will appear and user can’t modify data.
Patient Details:
On the basis of user selection patient details will appear and user can’t modify data.
Tissue/Organ:
User can write tissue and organism details, it can't remain blank.
Specimen Received By:
User can select the Specimen received by (Doctor) from the given list, this will come from employee screen, it can't remain blank.
Specimen Dispatched By:
User can select the Specimen dispatched by (Doctor) from the given list, this will come from employee screen, it can't remain blank.
Time of dispatch:
User can write Specimen dispatch details, it can't remain blank.
Clinical Notes:
User can write patient clinical note details, it can't remain blank.
Submit (Button):
By clicking on Submit button user can save the data.
Back (Button):
Clicking on Back button, will move user to the previous screen.
