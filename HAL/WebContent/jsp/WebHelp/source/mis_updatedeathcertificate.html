Update Death Certificate

Screen Overview
This screen will capture modify patient death details and it will issue modify death certificate.



Registration No:
User can write death certificate registration number based on registration number all patient death details will appear, it can't remain blank.
Service No:
Based on registration number service number will display User can’t modify service number, it can't remain blank.
HIN:
Based on registration number Hospital information number will display User can’t modify this number, it can't remain blank.
Ad No:
Based on registration number admission discharge number will display User can’t modify this number, it can't remain blank.
Patient Name:
Based on registration number patient name will display User can modify patient name, it can't remain blank.
Gender:
Based on registration number patient gender details will display User can modify patient gender, it can't remain blank.
Date of Death:
Based on registration number patient death date details will display User can modify patient death date, it can't remain blank.
Time of Death:
Based on registration number patient death time details will display User can modify patient death time date, it can't remain blank.
Mother’s name:
Based on registration number patient mother name details will display User can modify patient mother name, it can't remain blank.
Father’s name:
Based on registration number patient father name details will display User can modify patient father name, it can't remain blank.
Address of the deceased at the time of death:
Based on registration number address of deceased at the time of death will display User can modify this details, it can't remain blank.
Permanent address of deceased:
Based on registration number permanent address of deceased details will display User can modify this details, it can't remain blank
Date of Registration:
Based on registration number date of registration details will display User can modify date of registration, it can't remain blank.
Date of issuing:
Based on registration number date of certificate issue details will display User can modify date of issue, it can't remain blank.
Number of copies:
Based on registration number death certificate copies details will display User can modify number of copies required, it can't remain blank.
Amount:
Based on registration number amount details will display User can modify amount, it can't remain blank.
Remarks:
Based on registration number remarks details will display User can modify remarks, it can't remain blank.
Authenticated Employee:
Based on registration number authenticated employee details will display User can modify authenticated employee, it can't remain blank.
Submit (Button):
By clicking on Submit button user can modify the data.
