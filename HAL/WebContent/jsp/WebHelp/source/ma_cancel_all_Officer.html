Cancellation of Allotment Order for officers

Screen Overview
This screen will capture for cancellation of already allotment SMQ.



Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search. User can search HO_TO from date, HO_TO to date, Allotment order file number, service number, service type, rank, pool name and SMQ Number wise.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Go Button.
Cancellation:
Based on selection of allotment order file number user can cancel the already allotted SMQ.
Remarks:
User can write remarks and observations.
Save (Button):
By clicking on save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Can.Order (Button):
Clicking on cancel order button, user can print cancel order details.
