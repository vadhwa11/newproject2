IPD Discharge Sum NABH

Screen Overview
This screen captures the details of in patient NABH discharge summary details.



Patient Details:
On clicking of discharge summary button user can access discharge summary for Obstetric and gynecology cases. Based on patient selection from main screen patient details will display user canâ€™t modify patient details.
Select Dept:
User can select the discharge type from the given list, It can't remain blank. Based on selection of department type all department wise entries will display.
History with Reason for Admission:
User can write patient history with reasons details, it can remain blank.
Hospital Course:
User can write patient hospital course details, it can remain blank.
Investigation:
User can write patient investigation details, it can remain blank.
Procedure & Treatment Given:
User can write patient procedures and treatment given at the time of admission, it can remain blank.
When & how to obtain urgent care:
User can write patient urgent care details, it can remain blank.
Diagnosis:
Patient diagnosis details will display from patient admission details. it can remain blank.
Advise on Discharge (will be printed on Discharge slip also):
User can write patient advice on discharge details, it can remain blank.
Working Diagnosis:
User can write Patient working diagnosis details. it can remain blank.
Brief Case Summary (For Printing on Discharge slip):
User can write patient brief case summary details, it can remain blank.
Patient Diagnosis (Button):
By clicking on button user can enter patient diagnosis details.
Submit (Button):
By clicking on Submit button user can save the data.
Print (Button):
By clicking on print button user can take print out of discharge summary case sheet.
Discharge Slip (Button):
By clicking on discharge slip button user can take print out of discharge slip.
Back (Button):
Clicking on Back button, will move user to the previous screen.
