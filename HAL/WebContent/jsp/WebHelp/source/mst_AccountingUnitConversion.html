Accounting Unit Conversion Master

Screen Overview
This screen captures the details of item accounting unit conversion.



A/U Name:
On the basis of the search requirement, user can choose A/U Name.
Textbox and Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
Generate Report (Button):
By clicking on generate report button, report will be displayed.
A/U Name:
Enter A/U name. This is mandatory.
Purchase unit:
Select purchase unit from the list. This is mandatory.
PVMS No./NIV No.:
User can enter PVMS/NIV no. This is mandatory.
Conv. Factor 1:
User can enter conv. Factor1. This is mandatory.
Intermediate unit :
Select intermediate unit from the list. This is mandatory.
Conv. Factor 2:
User can enter conv. Factor2. This is mandatory.
Issue Unit:
Select Issue unit from the list. This is mandatory.
Formula:
Select formula conversion if item conversion is required else selects no conversion.
Add (Button) :
By clicking on add current screen data will be saved .
Reset (Button) :
By clicking on reset button all entered values will be removed.
