Edit Form Report

Screen Overview
This screen captures the user can edit or modify the user application details.



Application Name:
User can write application name details, based on user typing all application list will display user can select any one from the list and it can't remain blank.
Search (Button):
Based on selection of application name user can click on search button all details will display.
Parent Id:
Based on application name parent id will display if user wants to modify parent id user can modify.
Group Name:
Based on application name group name will display if user wants to modify the group name user can modify.
URL:
Based on application name URL will display if user wants to modify URL name user can modify URL.
Status:
Based on application name Status will display if user wants to modify status user can modify status.
Update (Button):
By clicking on Update button user can modify the data.
Back (Button):
Clicking on Back button, will move user to the previous screen.
