IPD Patient Remark

Screen Overview
This screen captures the patient remarks and treatment details.



Patient Details:
On clicking of patient issue button user can access patient issue screen for drug usage. Based on patient selection from main screen patient details will display user can’t modify patient.
Issue Date:
It is system generated date and present date and user can’t change date, it can’t remain blank.
Time:
It is system generated time and present time and user can’t change time, it can’t remain blank.
Condition:
User can select patient condition from the give list, it can’t remain blank.
Post op case:
User can select patient post operative case, It contains yes or no and it can’t remain blank.
Remarks:
User can write patient remarks details, it can remain blank.
Treatment:
User can write patient treatment details, it can remain blank.
Copy latest remarks & treatment (Button):
By clicking on Copy latest remarks & treatment button automatically previous details will appear.
Submit (Button):
By clicking on Submit button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Back (Button):
Clicking on Back button, will move user to the previous screen.
View :
By clUser can view select date and user can view the previous details in the grid.
