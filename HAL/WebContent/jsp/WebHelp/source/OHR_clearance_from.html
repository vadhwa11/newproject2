Posted Out Entry(Clearence Form)

Screen Overview
By using this screen user can do Posted Out entry.






Posted Out:
Search (Button):
User can search the records based on Service No,Rank,First Name,Last Name.Select the particuler search criteria and click on the search button.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Ok Button.
Employee Credential:
Posted Out Type:
user can select Posted Out Type from the drop down box,this is mandatory field.
Authority:
User can write Authority in this text box,this is mandatory field.
Date Of Posting Out:
User can write Date Of Posting Out in this text box,this is mandatory field.
SORS/SOS:
User can write SORS/SOS in this text box,this is mandatory field.
POR SL NO.:
User can write POR SL NO. in this text box,this is mandatory field.
Remarks:
User can write Remarks in this text box
For Officers:
Appraisal report/NIR last raised on:
User can write Appraisal report/NIR last raised on date in this text box
Submit (Button):
User can Save the record by clicking on the submit Button.
Reset (Button):
By clicking on this button all the text boxes will be clear.
Close (Button):
By clicking on this button page will be closed.
Search (Button):
By clicking on this button Search Posted Out Entry Page will be visible.
