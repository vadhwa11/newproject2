Work Order

Screen Overview
This screen captures the details of work order. When non expendable item requires repairing in the hospital work order is generated.



Work Order
Repairing Cell:
Select repairing cell from the list.
Hospital Name:
Hospital name will be displayed.
Work order:
This is auto generated no. by the system.
Date:
Current date will be displayed by the system. User can change this date.
Ward/Dept.:
Login department name will be displayed.
Authority No.:
User can write authority no.
Work order details:
SR No.:
Auto sequence no. will be generated by the system.
PVMS No.:
User can write PVMS No. and press tab. Details of item will be displayed.
Nomenclature:
It is description of PVMS No. If PVMS / NIV no. is not entered then type nomenclature initial 2-3 characters, list will be displayed. After nomenclature selection A/U will be displayed.
A/U:
Accounting unit of PVMS No will be displayed.
Serial No.:
User can select serial no from the list.
Quantity:
Write quantity.
Nature of work:
Write nature of work.
Remarks:
Write remarks.
Submit (Button):
By clicking on submit button data will be saved.
Search (Button):
By clicking on search button previous data will be search.
