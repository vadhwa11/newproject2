Update Supply Order Entry

Screen Overview
By using this screen user can update suppy order entry.Just click on the record you want to update then the corresponding records will be visible in the text boxes and click on the update button.




Supply Order Entry Search:
User can search supply orders based on Supply Order Number,Vendor Name,Quotation Name,Book Name And ACC number.
Supply Order No:
Based on selection of vendor Supply Order No will display, user can't modify.
Date:
Based on selection of vendor Date will display, user can't modify.
Vendor Name:
Based on selection of vendor Vendor Name Will display, user can modify the vendor.
Quotation No:
Based on selection of vendor Quotation Number Will display, user can modify.
Quotation Date:
Based on selection of vendor Quotation Date Will display, user can modify.
Remarks
Based on selection of vendor Remarks Will display, user can modify.
Add (Button):
User can Add a new row by clicking on this button to add the Books detail.
Delete (Button):
User can Delete the row by clicking on this button.
Book Namex:
User can update Book Name details.
Author:
User can update Author details.
Publisher:
User can update Publisher details.
Year Of Publication:
User can update Year Of Publication details.
Comparitive Cost:
User can update Comparitive Cost details.
Edition:
User can update book Edition details.
Update (Button):
By clicking on submit button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
