Tender Creation

Screen Overview
This screen captures the details of tender creation. Tender is created based on tender proposal or adding new item.



Group:
Select group name from the list for which tender has to create. If proposal is already created for selected group then “Import Proposals” button will be displayed.
Import Proposals (Button) :
By clicking on import proposals button , proposal data will be displayed in tender item details grid.
Tender No.:
This is yearly auto generated no by the system. Exp- 01/09-10,02/09-10 etc. This is mandatory.
Invit. Date:
Invitation date will be displayed by the system. User can change it. This is mandatory.
Commence Date:
commence date will be displayed by the system. User can change it. This is mandatory.
Last Date (issue):
Last date will be displayed by the system. User can change it. This is mandatory.
Time of receipt:
User can write time of receipt. This is mandatory
Place of receipt:
By default CHAF, Bangalore will be displayed. This is mandatory.
Last Date (receipt):
By default current date will be displayed. User can change Last date. This is mandatory.
Open Date (Tech.):
By default current date will be displayed. User can change open date of technical bid. This is mandatory.
Open Date (Comm.):
By default current date will be displayed. User can change open date of commercial bid. This is mandatory.
Terms & conditions:
User can write terms & conditions.
PVMS/NIV:
If user wants to search for any PVMS or NIV item then write PVMS/NIV code and click on search button.

Tender Item details:

SR No.:
Auto sequence no. will be generated by the system.
PVMS No.:
PVMS No. will be displayed by the system.
Nomenclature:
Nomenclature will be displayed by the system.
A/U :
Accounting unit of PVMS No will be displayed.
Stock in hand:
Login department Current stock will be displayed.
Tender qty. (Exp. Store):
Proposal qty. will be displayed for expendable store.
Tender qty. (ECHS):
Proposal qty. will be displayed for ECHS store.
Tender qty. (Total):
Sum of tender qty. for both expendable & ECHS store will be displayed.
Quantity :
Available quantity will be displayed.
Create (Button):
By clicking on create button tender will be created.
Add (Button):
By clicking on Add button selected tender data will be displayed in grid. User can select item, change qty.
Update (Button):
By clicking on Update button changed data will be update.
Delete (Button):
By clicking on Delete button selected row in grid will be deleted.
Print (Button):
By clicking on Print button Proposal will be printed.
