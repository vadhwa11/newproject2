Condemnation Entry

Screen Overview
This screen captures the details of item condemnation. When item is not repairable & not usable then condemnation entry is done.



Details
Condemnation No.:
This is auto generated no. by the system.
Condemnation Date:
Current date will be displayed by the system. User can change this date.
Work order no.:
Select work order no. from the list. This is filled from Issue CIV entry.
Work order date:
Based on work order no selection date will be displayed.
Work order:
SR No.:
Auto sequence no. will be generated by the system.
PVMS No.:
PVMS no. will be displayed by the system.
Nomenclature:
Nomenclature will be displayed by the system.
Serial No.:
Serial no will be displayed by the system.
Part No.:
Part no will be displayed by the system.
Quantity:
Quantity will be displayed by the system.
Reason for sentence:
Write reason for sentence.
Submit (Button):
By clicking on submit button data will be saved.
Search (Button):
By clicking on search button previous data will be search.
