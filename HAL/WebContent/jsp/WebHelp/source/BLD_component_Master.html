Blood Component Master

Screen Overview
By using this screen user can add and edit the Blood Component.For update just click on the particuler record you want to update then the corresponding records will be visible in the text boxes.



Component Code or Component Name :
On the basis of the search requirement, user can either choose Component code or Component name.
Textbox and Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Go Button.
Component Code Text Box:
User can write component code details, it can't remain blank.
Component Name Text Box:
User can write component name details, it can't remain blank.
Life span Text Box:
User can write life span details and select days and months from the given list.
Kept at Temp Text Box:
User can write the temperature in Â°C details,it can remain blank
Default qty. per unit Text Box
User can write default quantity/Unit details, it can't remain blank
Whole Blood Drop down
User can select whole blood yes/no from the given list,it can remain blank
Add (Button):
By clicking on Add button user can Add the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Update (Button):
By clicking on Update button user can Update the data.
Inactivate (Button):
By clicking on inactive button user can inactive the existing data.
