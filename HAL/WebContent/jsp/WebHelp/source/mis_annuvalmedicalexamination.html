Annual Medical Examination

Screen Overview
This screen captures employee annual medical examination details and it will capture next employee review date details.



Service No:
User can write employee service number, it can't remain blank.
Rank:
Based on service number employee service person rank will display, it can't remain blank.
Name:
Based on service number employee service person name will display, it can't remain blank.
Trade:
Based on service number employee service person trade will display, it can remain blank.
Parent Unit:
Based on service number employee present unit name will display, it canâ€™t remain blank.
Medical Category:
User can select the medical category from the given list, it can't remain blank.
Period:
User can select the period from the given list, it contains Permanent and temporary, it can't remain blank.
Last AME/Board:
User can select the last annual medical exam date details, it can't remain blank.
Next Review Date:
User can select the employee next review date details, it can't remain blank.
Submit (Button):
By clicking on Submit button user can save the data.
