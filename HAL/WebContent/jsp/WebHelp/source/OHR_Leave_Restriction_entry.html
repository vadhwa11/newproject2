Leave Restriction entry

Screen Overview
By using this screen user can do Leave Restriction entry.






Entry No.:
User can write Entry No. in this text box.
Entry Date:
User can write Entry Date in this text box.
From Period:
User can write From Period in this text box.
To Period:
User can write To Period in this text box.
Leave Type:
User can select leave type from the drop down box.
Max Leave Days:
User can write Max Leave Days in this text box.
Remarks:
User can write Remarks in this text box.
Submit (Button):
By clicking on Submit button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Close (Button):
To close this screen click on this button.
Search (Button):
Clicking on search button Leave Restriction Search page will be visible.
