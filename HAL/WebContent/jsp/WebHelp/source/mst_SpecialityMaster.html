Speciality Master

Screen Overview
This screen captures the Specialty details.



Specialty Code or Specialty Name:
On the basis of the search requirement, user can either choose Specialty code or Specialty name.
Textbox and Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
Go Button and Text Box :
If user wants to switch directly to any particular page no., the he can simply enter the page no. in the given text box and click on Go Button.
Specialty Code:
User can write Specialty code details, it can't remain blank.
Specialty Name:
User can write Specialty name, it can't remain blank.
Save (Button):
By clicking on Save button user can save the data.
Update (Button):
By clicking on Update button user can modify the existing data.
Inactive (Button):
By clicking on inactive button user can inactive the existing data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Back (Button):
Clicking on Back button, will move user to the previous screen.
