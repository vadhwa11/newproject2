Investigation Appointments SetUp

Screen Overview
This screen captures the Investigation Appointments setup information for the department wise investigations. This screen is used for maintaining investigation appointments setup based on department available days. This screen will contain information of Week Days, No of Equipments , From time, To time, Slot duration, Break Time, Max no of days and Min no of days



Department:
User can select the department from the given list, this will come from department screen, it can't remain blank.
Equip.Invesigation:
User can select the equipment from the given list, this will come from OPD equipment screen,it will diaply based on department and it can't remain blank.
From Time:
OPD starting time, it can't remain blank.
To Time:
OPD closing time, it can't remain blank.
Slot Duration:
Slot duration of each Patient, it can't remain blank.
Break From Time:
Break from time between from time and to time, it can remain blank.
Break To Time:
Break to (Closing) time between from time and to time, it can remain blank.
Max No. of days:
User can enter maximum number of days for giving OPD appointments, it can remain blank.
Min No. of days:
User can enter minimum number of days for giving OPD appointments, it can remain blank.
Submit (Button):
By clicking on Submit button user can save the data.
Reset (Button):
Clicking on Back button, will move user to the previous screen.
