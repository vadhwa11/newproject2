Blood Request Entry

Screen Overview
By using this screen user can do blood request entry.Just search your data and you will get the records into grid and click on the particular record for which you need to do request.






Search (Button):
User can search the records based on the HIN No.,Service No.,Patient Type,Patient First Name, Patient Last Name, Rank.Just write your search criteria and click on the search button.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Go Button.
Service Personnel Details:
This section will give the details of the record selected by you from the grid like - Order Number,Order Date,Service No.,HIN No.,Rank.
Patient Details:
This section will give the detail of the patient like Patient Name, Age, Blood Group.
Request Type Drop Down Box:
User can select the request type from the give list, it can remain blank.
Date & no. of bottles given Text Box:
User can write the date and number of bottles details, it can remain blank.
Hb % Text Box
User can write Hb % details, it can remain blank.
Presence:
User can write presence details, it can remain blank.
Fever Text Box:
User can write Fever details, it can remain blank.
Of Time Text Box:
User can write of time details, it can remain blank.
History of Pregnancies(if any) Text Box:
User can write History of Pregnancies(if any) details , it can remain blank.
With specific reference and diagnostic disease of new born Text Box:
User can write the details of the "With specific reference and diagnostic disease of new born",, it can remain blank.
GRID:
User can write the details of the Component Name,Qty(ml),Required on Date in the corresponding details , it can remain blank.
Add (Button):
By clicking on Add button user can Add the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
