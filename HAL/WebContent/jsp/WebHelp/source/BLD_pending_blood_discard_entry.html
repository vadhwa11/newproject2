Pending For Blood Discard

Screen Overview
By using this screen user can see the pending for blood discard records, just click on the record for which you want to do blood discard entry then blood discard entry page will be visible.






Pending For blood Discard:
Grid:
User can see the records which are pending for blood discard.Juat click on the record from the grid for which you would like to do blood discard entry.

Blood Discard Entry Page:
This page will show the details of the record selected by user from the grid like- discard no,Blood bag no.,date of collection,blood component name,Quantity,rank,Unit/Address,Date of expiry,Blood Group.
Discard Date:
User can write discard date details.
Approved By:
User can select approved by person from the given list, it can't remain blank.
Remarks:
User can write remarks details.
Service No.:
User can write Service number details.
Submit Button:
By clicking on the submit button user can save the record.
Reset Button:
Clicking on reset button, will clear the current or existing data.
