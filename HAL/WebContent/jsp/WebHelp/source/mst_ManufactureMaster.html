Manufacture Master

Screen Overview
This screen captures the details of manufacturer entry.



Manufacturer code/Manufacturer name :
Select manufacturer code or manufacturer name for search.
Search (Button):
By clicking on search button, entered manufacturer details data will be displayed.
Manufacturer Code:
This is auto code generated by the system. Exp- M1, M2 etc. This is mandatory.
Manufacturer Name:
User can enter manufacturer name. This is mandatory.
Contact Person:
User can write contact person.
C&F/ Local distributor name:
User can write C&F local distributor name.
C&F/ Local distributor address1:
User can write C&F local distributor address1.
C&F/ Local distributor address2:
User can write C&F local distributor address2.
Address1:
User can write address1.
Address2 :
User can write address2.
State:
User can select state.
City/District:
User can select city/District.
Phone no.:
User can write phone no.
Mobile no.:
User can write mobile no.
PIN code no:
User can write Pin code no.
Email id:
User can write email id.
Fax no.:
User can write Fax no.
URL:
User can write URL.
License no.:
User can write license number.
Sales Tax no.:
User can write sales tax number.
Add (Button) :
By clicking on add current screen data will be saved.
Reset (Button) :
By clicking on reset button all written values will be removed.
