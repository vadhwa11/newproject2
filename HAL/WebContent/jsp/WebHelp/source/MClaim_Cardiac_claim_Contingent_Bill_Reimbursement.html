Cardiac claim-Contingent Bill for Reimbursement

Screen Overview
By using this screen user can write detail of Cardiac claim-Contingent Bill for Reimbursement.






Search (Button):
User can search the records based on the Service No,Hin,Patient First Name,Patient Last Name,Ser. First Name,Ser. Last Name,Rank(Select from the Drop Down Box), Just write your search criteria and click on the search button.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on OK Button.
Cardiac claim - Contingent Bill for Reimbursement page:
This page will show the details of the record selected by user from the GRID like- Entry No,Entry Date,Service No.,Name,Rank,Branch/Trade,Unit,Unit,Address etc.
(+) Button :
User can add new row by clicking on this button to enter more Bill Details.
(-) Button:
User can delete the row by clicking on this button.
Details Text Box:
User can write bill details into this text box.
Bill No Text Box:
User can write Bill No into this text box, this is mendatory field.
Bill Date Text Box:
User can write bill date into this text box.
Amount Text Box:
User can write Amount details into this text box.
Amount qualifying for Text Box:
User can write Amount qualifying for details into this text box, this is mendatory field.
Reimbursement Rs Text Box:
User can write Reimbursement Rs details into this text box,this is mendatory field.
Amount Payable to Text Box:
User can write Amount Payable to details into this text box,this is mendatory field.
The CGHS code for Text Box:
User can write The CGHS code for details into this text box,this is mendatory field.
is Text Box:
User can write is details into this text box,this is mendatory field.
CGHS rates in drop down box Text Box:
User can select CGHS rates in details from the drop down box,this is mendatory field.
is Text Box:
User can write is details into this text box,this is mendatory field.
Received (Rs.) Text Box:
User can write Received (Rs.) details into this text box,this is mendatory field.
Submit (Button):
By clicking on Add button user can Add the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
