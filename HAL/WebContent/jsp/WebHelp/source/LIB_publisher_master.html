Publisher Master

Screen Overview
This screen have list of Publisher code along with Publisher names and status.User can add and update the Publisher details by using this screen.



Publisher Code or Publisher Name:
On the basis of the search requirement, user can either choose Publisher Code or Publisher Name
Search (Button):
Textbox is to write the value you are searching or looking for. Once it's done you can click on search button to get the result of your search.
GRID (Display):
Based on search criteria data will display in the form of table. At a time only five (5) records will be displayed. To navigate or look for the next record pagination (the system of numbering pages) will be there.
OK Button and Text Box:
If user wants to switch directly to any particular page no, then he can simply enter the page no. in the given text box and click on Go Button.
Publisher Code:
User can write the Publisher Code details, it can't remain blank.
Publisher Name Text Box:
User can write the Publisher Name details, it can't remain blank.
Address(Line1)Text Box:
User can write the part of the Address details, it can't remain blank.
Address(Line2)Text Box:
User can write the part of the Address details, it can remain blank.
Address(Line3) Text Box:
User can enter the part of the Address details, it can remain blank.
City:
User can select city from the given list, it can remain blank.
State:
User can select the state from the given list, it can remain blank.
Pin Text Box:
User can write the Pin code details, it can remain blank.
Add (Button):
By clicking on Add button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
Update (Button):
By Clicking on Update button, user can update the data.
Inactivate (Button):
By Clicking on Inactivate button, user can inactivate any machine item.
