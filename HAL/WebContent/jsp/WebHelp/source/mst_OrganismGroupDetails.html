Organism Group Details Master

Screen Overview
This screen captures the microbiology organism group and organism details.



Organism Group:
User can select the organism group from the given list, this will come from organism group screen and it can't remain blank. Based on selection of organism group all organism list will display. User can add more.
GRID (Display):
Based on organism group selection data will display in the form of table. At a time only ten records will be displayed. If user wants more rows user can press + symbol then automatically one row will created. User can select organism from the list user can write any character based on that list will display. User can select any one.
Submit (Button):
By clicking on Save button user can save the data.
