Assign Module to Employee Group

Screen Overview
This screen captures the assign total module to all employee groups.



Hospital Name:
Default login hospital name will display. User can select the hospital name from the given list, this will come from hospital screen, it can't remain blank.
Application group:
User can select the application group from the given list, this will come from application group screen, it can't remain blank.
Application grid:
Based on user selection of application group all applications list will display. User can select one or multiple application and assign to total employee group.
Employee Group:
User can select the employee group from the given list, this will come from employee group screen, it can't remain blank.
Users grid:
Based on user selection of employee group all users list will display. User can select one or multiple users and assign the rights. User wants see the present rights for particular user and select option button then user can view the present rights.
Assign Module (Button):
Clicking on Assign ,module button, all modules will assign to selected user.
Back (Button):
Clicking on Back button, will move user to the previous screen.
