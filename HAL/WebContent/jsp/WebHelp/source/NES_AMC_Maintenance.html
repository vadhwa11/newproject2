AMC Maintenance

Screen Overview
This screen captures the details of AMC maintenance.



Department name:
Select department name from the list where item is used. This is mandatory.
Nomenclature:
Select department name from the list where item is used. Based on nomenclature serial no. will be filled. This is mandatory.
PVMS/NIV:
PVMS/NIV no. will be displayed based on nomenclature.
Serial No.:
Select serial no. from the list.
Doc entry No.:
Write doc entry no.
First AMC start date:
First AMC start date will be displayed by the system.
CRV No.:
CRV no. will be displayed by the system.
Entry date:
By default current date will be displayed.
SO No.:
SO no. will be displayed by the system.
Warranty start date:
Warranty start date will be displayed by the system.
Warranty end date:
Warranty end date will be displayed by the system.
Cost of equipment:
Cost of equipment will be displayed by the system.
Support start date:
User can write support start date.
Support end date:
User can write support end date.
Date of installation:
Date of installation will be displayed by the system.
Total received qty.:
Total received qty will be displayed by the system.
AMC Details:
AMC company:
Select AMC company name from the list.
AMC date from:
User can select AMC date from.
AMC date to:
User can select AMC date to.
Cost of AMC:
User can write cost of AMC.
Adv. Bill no.:
User can write advance bill no.
Adv. Bill date:
User can write advance bill date.
Adv. Bill amt.:
User can write adv. Bill amt.
Balance Bill no.:
User can write balance bill no.
Bal. Bill date:
User can select balance bill date.
Bal. Bill amt.:
User can write balance Bill amt.
Remarks:
Write remarks.
Add Row (Button):
By clicking on add row button new row will be created.
Submit (Button):
By clicking on submit button data will be saved.
