Marriage Accommodation Registration for Airmen

Screen Overview
This screen will capture accommodation registration for airmen, it contains airmen service details and AME details.



Registration No:
It is system generation number, user can’t change number.
Registration Date:
It is system generated date, user can’t change date and it is present date.
Registration Time:
It is system generated time, user can’t change time and it is present time.
Unit:
User can select the unit from the given list, this will come from unit screen, and it can't remain blank. If unit is “CHAFB” then Service personal details will come from automatically like Service no, service person name, rank, trade and sfx. If Unit is “CHAFB” then all Service personal details user should write. Like Service no, service person name, rank, trade and sfx.
Date of Reporting:
It is present date and user can change date.
Previous Unit:
User can select the previous unit from the given list, this will come from unit screen, it should not same as unit. And it can't remain blank.
Pool Code:
User can select the pool code from the given list, this will come from pool code screen, it will display only airmen pools and rank wise .It can't remain blank.
Previous SMQ:
User can select option retaining or not retaining. It can't remain blank.
Date of AME:
User can write his marriage date. It can't remain blank.
Remarks:
User can write his remarks and observations.
Amount:
User can write amount details, it can't remain blank.
Anti-Date Seniority:
User can write his Seniority date. It can't remain blank.
Anti-Date Remarks:
User can write his remarks and observations.
Save (Button):
By clicking on save button user can save the data.
Reset (Button):
Clicking on reset button, will clear the current or existing data.
