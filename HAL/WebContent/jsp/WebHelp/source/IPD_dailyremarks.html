IPD Daily Remarks

Screen Overview
This screen captures the details of daily patient remarks entry details based on selected patient.



Date:
It is default system date, user can modify this date.
Time:
It is system time, user can’t modify time. e.
Remarks:
User can write present patient remarks or observations, it can’t remain blank.
Submit (Button):
By clicking on Submit button user can save the data.
Back (Button):
Clicking on Back button, will move user to the previous screen.
Date:
It is system date, user can change date and user can view previous date details.
View (Button):
By clicking on date user can view the patient previous remarks details.
