Complaint Monitoring System

Module Overview
Command Hospital Air Force Bangalore (CHAFB) needs to maintain the Work Services, online Complaint Register and Agenda points for work services records etc. efficiently at their hospital. At any given time these Module should be able to get the correct information from each of their location.

In this Module has following Sub Modules
  1. Complaint Type Master
  2. Complaints Register
  3. Work Completion Details
  4. Work Not Completed With In 3 Days
